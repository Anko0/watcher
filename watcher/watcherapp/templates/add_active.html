Add new active
How to add a new active? Read the instruction:
  1. Copy project directory 'agent' to the remove host (your new active)
  2. Run installer.sh as root
  3. Target watcher_agent.json is /opt/watcher-agent/watcher_agent.json, edit it:
    1. Copy 'Token for API' from the form below and paste its value on ACTIVE_TOKEN variable in watcher_agent.json
    2. Fill API_URL variable in watcher_agent.json (format: http(s)://watcher_ip:watcher_port/metrix/)
    3. Fill TIME_DELAY variable in watcher_agent.json (means how often to collect metrics)
    4. Save the file
  4. Complete the form below
  5. Click Save
  6. Run the watcher agent systemd service on the host
